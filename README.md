
## Installation

1. Clone the repository:

   ```
   git clone https://github.com/Rahulfordev/Balance-Calculator.git
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. Run the application:

   ```
   npm run dev
   ```

4. Open [http://localhost:5173/](http://localhost:5173/) to view it in the browser.
